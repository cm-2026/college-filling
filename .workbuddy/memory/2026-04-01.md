### 修复关键Bug

**问题**: 用户列表查询报错 "Incorrect arguments to mysqld_stmt_execute"

**原因**: MySQL2的prepared statement不支持LIMIT和OFFSET参数绑定

**解决方案**: 使用模板字符串直接拼接LIMIT和OFFSET值
```javascript
// 错误写法
sql += ' LIMIT ? OFFSET ?';
params = [...params, pageSize, offset];

// 正确写法
sql += ` LIMIT ${pageSize} OFFSET ${offset}`;
// 不传递LIMIT和OFFSET参数
```
