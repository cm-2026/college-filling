## 三科选科要求查询修复（简化版）

**问题**：数据库中三科要求只有"物理和化学和生物"这一种形式，没有"+"连接的数据

**修改内容**：
- 简化`server.js`中的三科匹配逻辑
- 只保留"和"分隔符，移除"+"分隔符支持
- 减少12种匹配格式 → 6种匹配格式

**修改位置**：
1. 3+3模式：server.js 第325-333行
2. 3+1+2模式：server.js 第474-485行

**技术实现**：
```javascript
// 只生成6种排列组合
const triples3 = [];
for (let i = 0; i < subjects.length; i++) {
  for (let j = 0; j < subjects.length; j++) {
    for (let k = 0; k < subjects.length; k++) {
      if (i !== j && i !== k && j !== k) {
        triples3.push(`${subjects[i]}和${subjects[j]}和${subjects[k]}`);
      }
    }
  }
}
```
