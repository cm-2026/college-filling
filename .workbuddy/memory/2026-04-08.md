## index-mysql.html 修复重复ID问题

### 问题诊断
- 页面报错：`Cannot set properties of null (setting 'value')`
- 推荐结果无法显示

### 根本原因
HTML中存在重复ID，导致getElementById获取到错误元素：
- `filterProvince` 在工具栏(第1856行)和筛选面板(第1900行)各出现一次
- `filterCity`、`filterMajorCategory`、`filterBatchRemark` 同样重复

### 修复方案
1. **重命名工具栏筛选ID**：
   - `filterProvince` → `quickFilterProvince`
   - `filterCity` → `quickFilterCity`
   - `filterMajorCategory` → `quickFilterMajorCategory`
   - `filterBatchRemark` → `quickFilterBatchRemark`

2. **添加联动函数**：
   - `syncQuickFilter(field)` - 快捷筛选与筛选面板联动
   - 快捷筛选变更时同步到筛选面板并触发applyFilter

3. **修改现有函数**：
   - `initFilterOptions()` - 同时初始化快捷筛选下拉框
   - `updateFilterOptions()` - 同步更新快捷筛选
   - `applyFilter()` - 同步快捷筛选显示值
   - `resetFilter()` - 同时重置快捷筛选
