### 位次查询范围修复（2026-04-11 15:55）

**问题**：河南物理类600分（位次45887），数据库查询179所学校，但页面返回1981所

**根本原因**：
- 查询条件只有 `min_rank_1 >= userRank`（无上限）
- 导致查询所有录取位次高于用户的学校（数量过多）

**修改**：
```javascript
// 原逻辑：只查下限
whereCondition = `AND min_rank_1 >= ?`;
rangeParams = [userRank];

// 新逻辑：添加上限（上浮26%）
const maxRank = Math.round(userRank * 1.26);
whereCondition = `AND min_rank_1 >= ? AND min_rank_1 <= ?`;
rangeParams = [userRank, maxRank];
```

**效果**：位次45887 → 查询范围45887~57818，返回结果接近预期数量
