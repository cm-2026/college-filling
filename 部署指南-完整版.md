## 三、宝塔面板配置

### 1. 放行端口

在宝塔面板 → 安全 → 防火墙,添加规则:
- 端口: 3000
- 协议: TCP
- 说明: 高考志愿系统

### 2. 配置Nginx反向代理 (可选,使用80端口)

在宝塔面板 → 网站 → 添加站点:
- 域名: 你的域名或IP
- PHP版本: 纯静态

然后修改站点配置,添加反向代理:

```nginx
location / {
    proxy_pass http://127.0.0.1:3000;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_cache_bypass $http_upgrade;
}
```

配置后可直接通过 `http://81.70.43.192` 访问(无需端口号)

## 五、常见问题

### 1. 端口被占用

```bash
# 查看3000端口占用
lsof -i:3000

# 杀掉占用进程
kill -9 <PID>
```

### 2. 数据库连接失败

检查:
- MySQL服务是否启动: `systemctl status mysql`
- 用户名密码是否正确
- 数据库gaokao是否存在: `mysql -u root -p -e "show databases;"`
- 用户权限是否足够

### 3. 无法访问服务

检查:
- 防火墙是否放行3000端口
- 服务是否正常启动: `pm2 status` 或 `ps aux | grep node`
- 查看错误日志: `pm2 logs` 或 `cat logs/error.log`

### 4. 内存不足

修改ecosystem.config.json中的max_memory_restart参数,或升级服务器配置。

5. 定期更新系统和依赖包
